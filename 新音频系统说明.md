## 🔧 工作原理

### 浏览器自动播放限制
现代浏览器（Chrome、Firefox、Safari）有严格的自动播放政策：
- ❌ 不允许网页在没有用户交互的情况下自动播放音频
- ❌ 这是防止网页扰民的保护机制

### 我们的解决方案
```javascript
// 1. 监听所有用户交互事件
document.addEventListener(['click', 'keydown', 'touchstart', ...], () => {
  // 2. 标记用户已交互
  hasUserInteracted = true;

  // 3. 立即播放背景音乐
  if (soundEnabled && !isMusicPlaying) {
    backgroundMusic.play();
  }
});
```

### 技术细节
- **预加载机制**：页面加载时预加载音频文件
- **AudioContext激活**：用户交互时激活音频上下文
- **错误处理**：优雅处理播放失败的情况
- **状态同步**：音乐指示器实时反映播放状态

### 常见问题

**Q: 为什么第一次点击页面没有音乐？**
A: 检查音效开关是否开启，查看控制台是否有错误信息。

**Q: 音乐播放了一会就停止了？**
A: 不应该发生。如果发生，可能是音频文件加载失败，检查网络连接。

**Q: 音乐指示器不显示？**
A: 确认音效开关已开启，且音乐正在播放。
