# 贪吃蛇游戏 - 音效和特效功能说明

## 🎮 新增功能

### 1. 背景音乐 🎵
- 游戏开始时自动播放背景音乐
- 使用免费的 Pixabay 音乐资源
- 音量适中，不会��扰游戏体验
- 可通过音效开关控制

### 2. 吃果子音效 🔊
- 使用 Web Audio API 生成悦耳的和弦音效
- 每次吃到果子时播放大三和弦（C5-E5-G5）
- 音效可以根据用户偏好开启/关闭

### 3. 视觉特效 🎆
吃到果子时的三重特效：
- **粒子爆炸**：12个彩色粒子从果子位置向四周扩散
- **光环扩散**：绿色光环从中心向外扩散消失
- **分数飘字**："+10" 分数向上飘起并放大消失

### 4. 音效控制开关 🎛️
- 在游戏控制面板中添加了音效开关
- 用户可以自由选择是否开启音效
- 设置会保存在本地浏览器中

## 🔧 技术实现

### 音效系统
- 使用 Web Audio API 生成实时音效
- 双振荡器创建丰富的和弦效果
- 音量控制和淡出处理
- 性能优化，不影响游戏流畅度

### 视觉特效
- CSS 动画和 JavaScript 动画结合
- 粒子系统使用 DOM 元素和 Web Animations API
- 自动清理，避免内存泄漏
- 响应式设计，适配不同屏幕尺寸

### 用户体验
- 音效设置本地存储
- 平滑的动画过渡
- 清晰的视觉反馈
- 可访问性设计

## 🎯 使用方法

1. **打开游戏**：访问部署的网站或本地打开 index.html
2. **开始游戏**：点击"开始游戏"按钮或按 Enter 键
3. **体验音效**：
   - 背景音乐会开始播放
   - 吃到果子时会听到和弦音效
   - 看到粒子爆炸、光环扩散和分数飘字效果
4. **控制音效**：使用音效开关来开启/关闭所有音效

## 🎨 特效详情

### 粒子特效
- 12个粒子，随机大小和颜色
- 爆炸式扩散效果
- 800ms 动画时长
- 使用缓动函数实现自然的运动

### 光环效果
- 绿色光环从中心扩散
- 带阴影效果，增强视觉冲击
- 600ms 动画时长
- 逐渐放大并消失

### 分数飘字
- 绿色 "+10" 文字
- 向上飘动并放大
- 700ms 动画时长
- 带发光效果

## 🎵 音效详情

### 背景音乐
- 来源：Pixabay 免费音乐库
- 格式：MP3
- 循环播放

### 吃果子音效
- 实时生成，无需外部文件
- 大三和弦（C5-E5-G5）
- 150ms 持续时间
- 音量包络控制
