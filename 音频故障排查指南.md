## 🔍 故障排除步骤

### 步骤1：基础检查
- [ ] 确认音效开关是**开启**状态
- [ ] 确认电脑/手机音量没有静音
- [ ] 尝试刷新页面（F5）

### 步骤2：使用测试按钮
1. 点击 **"测试音频"** 按钮
2. 查看弹窗提示：
   - ✅ **测试成功**：音频系统正常，问题可能是自动播放限制
   - ❌ **测试失败**：查看错误信息

### 步骤3：查看控制台
在控制台中查找这些关键信息：

**成功日志**：
```
🎵 初始化音频系统...
✅ 背景音乐加载完成，可以播放
📁 当前音频源: [URL]
✅ 背景音乐开始播放
```

**失败日志**：
```
❌ 背景音乐加载失败: MediaError
💡 建议：检查网络连接或更换音频源
```

### 步��4：常见错误处理

#### 错误1：NotAllowedError
**症状**：浏览器阻止了自动播放
**原因**：浏览器自动播放政策
**解决方案**：
1. 点击页面任何位置
2. 点击"测试音频"按钮
3. 或者直接点击"开始游戏"

#### 错误2：NotSupportedError
**症状**：音频格式不支持
**原因**：所有音频源都失败
**解决方案**：
1. 检查网络连接
2. 尝试使用不同的浏览器
3. 联系我更换其他音频源

#### 错误3：网络错误
**症状**：音频文件加载失败
**原因**：网络连接问题
**解决方案**：
1. 检查网络连接
2. 刷新页面重试
3. 清除浏览器缓存

## 🧪 高级调试

### 手动测试音频（在控制台执行）

```javascript
// 测试1：检查音频元素
const audio = document.getElementById('backgroundMusic');
console.log('音频源:', audio.src);
console.log('就绪状态:', audio.readyState);
console.log('网络状态:', audio.networkState);

// 测试2：手动播放
audio.volume = 0.5;
audio.play().then(() => {
  console.log('✅ 播放成功');
}).catch(err => {
  console.error('❌ 播放失败:', err);
});

// 测试3：检查所有音频源
const sources = audio.querySelectorAll('source');
sources.forEach((source, index) => {
  console.log(`音频源${index + 1}:`, source.src);
});
```
